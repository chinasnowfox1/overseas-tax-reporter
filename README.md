# 境外投资报税助手 (Overseas Tax Reporter)

> Claude Code Skill — 中国居民个人境外投资年度个税申报全流程自动化

## 一句话介绍

从一堆券商PDF月结单开始，到生成符合律师模板的7-sheet报税Excel结束，全程引导+自动计算，不用懂税法。

---

## 适用人群

- 中国税务居民，有境外投资（股票、基金、利息等）
- 每年需要申报境外收入个税，但不知道从何开始
- 不想手工整理券商月结单到Excel

## 安装

```bash
git clone https://github.com/chinasnowfox1/overseas-tax-reporter.git ~/.claude/skills/overseas-tax-reporter
```

然后在 Claude Code 里说 **"我要报境外税"** 即可触发。

## 依赖

```bash
pip install pdfplumber pandas openpyxl
```

---

## 工作流程

```
原始PDF月结单
    │
    ▼
PDF解析 (parser.py) → 结构化交易数据
    │
    ▼
税务计算 (calculator.py) → 盈亏互抵 + 预扣抵免
    │
    ▼
Excel输出 (exporter.py) → 7-sheet律师模板
    │
    ▼
用户校验 → 提交给税务律师
```

### 7阶段引导

| 阶段 | 做什么 | 输出 |
|------|--------|------|
| 0 | 确认纳税年度 | 年份 |
| 1 | 收集原始素材 | 文件清单 |
| 2 | 税法规则科普 | 收入类型+税率 |
| 3 | PDF解析 | 结构化交易数据 |
| 4 | 税务计算 | 应纳税额 |
| 5 | 生成Excel | 7-sheet报税文件 |
| 6 | 用户校验 | 确认或修正 |

---

## 税法规则速查

| 收入类型 | 税率 | 规则 |
|----------|------|------|
| 股票买卖盈亏 | 20% | 同年度盈亏互抵，不跨年结转 |
| 基金赎回盈利 | 20% | 同上 |
| 股息/分红 | 20% | 境外预扣可抵免 |
| 银行存款利息 | 20% | 境外银行利息不免税 |

完整规则见 [references/tax_rules.md](references/tax_rules.md)。

---

## 输出格式

生成7个sheet的Excel，严格对齐税务律师要求的模板：

| Sheet | 内容 |
|-------|------|
| 股票基本信息 | 持仓证券代码、名称、注册地 |
| 股票交易明细 | 买卖方向、数量、价格、盈亏 |
| 股息 | 分红日期、税前金额、预扣税 |
| 利息收入 | 银行利息 |
| 基金交易 | 申购/赎回明细 |
| 税款计算表 | 分类汇总+汇率换算+税额 |
| 透视表 | 交叉汇总 |

|------|--------|------|
| 富途证券 | ✅ 已验证 | 完整PDF解析（股票/股息/基金） |

| 其他券商 | 🔄 可扩展 | 提供月结单后可适配 |

## 隐私保护

- Skill代码中**不含任何用户个人数据**
- 所有示例使用占位符（`[STOCK_CODE]`、`[AMOUNT]`等）
- PDF密码通过用户输入获取，不写入代码

---

## 使用限制

- 目前仅支持年度汇算（自然年1月1日-12月31日）
- 财产转让所得按次征收，亏损不可结转至下一年度
- 汇率使用纳税年度最后一日（12月31日）的中国人民银行中间价
