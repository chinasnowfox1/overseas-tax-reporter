# Excel模板说明

## 输出格式: 7个Sheet

### Sheet 1: 股票基本信息
| 列名 | 说明 |
|------|------|
| 证券代码 | 如 [STOCK_CODE] |
| 证券名称 | 如 [STOCK_NAME] |
| 注册地 | 如 [REGION] |
| 是否H股 | 否 |
| 纳税年度是否有错配 | 否 |

### Sheet 2: 股票交易明细
| 列名 | 说明 |
|------|------|
| 证券代码 | [STOCK_CODE] |
| 证券名称 | [STOCK_NAME] |
| 注册地址 | [REGION] |
| 资金收付日期 | [DATE] |
| 币种 | [CURRENCY] |
| 本次买入股数 | 买入时填写 |
| 本次支出 | 买入时填写 |
| 本次卖出股数 | 卖出时填写 |
| 本次收入 | 卖出时填写 |
| 累计股数 | 0 |
| 累计支出 | 0 |
| 本次交易后加权平均持股成本 | [PRICE] |
| 本次卖出盈利 | 盈亏金额 |
| 手续费 | [AMOUNT] |
| 纳税年度是否错配 | 否 |
| 境外纳税年度 | [YEAR] |
| 对应内地纳税年度 | [YEAR] |

### Sheet 3: 股息
| 列名 | 说明 |
|------|------|
| 证券代码 | [STOCK_CODE] |
| 证券名称 | [STOCK_NAME] |
| 注册地 | [REGION] |
| 是否H股 | 否 |
| 分红日期 | [DATE] |
| 分红币种 | [CURRENCY] |
| 实际到账分红金额 | 税前 - 预扣 |
| 已代扣代缴税款币种 | [CURRENCY] |
| 手续费 | 0 |
| 已代扣代缴税款金额 | [AMOUNT] |
| 还原税前分红金额 | 税前总额 |
| 纳税年度是否错配 | 否 |
| 境外纳税年度 | [YEAR] |
| 对应内地纳税年度 | [YEAR] |
| 已扣税税率 | 预扣/税前 |

### Sheet 4: 利息收入
| 列名 | 说明 |
|------|------|
| 利息收取日期 | [DATE] |
| 币种 | [CURRENCY] |
| 利息金额 | [AMOUNT] |
| 纳税年度是否错配 | 否 |
| 境外纳税年度 | [YEAR] |
| 对应内地纳税年度 | [YEAR] |

### Sheet 5: 基金交易
| 列名 | 说明 |
|------|------|
| 名称 | [FUND_NAME] |
| 注册地址 | [REGION] |
| 币种 | [CURRENCY] |
| 资金收付日期 | [DATE] |
| 本次买入数量 | 申购时填写 |
|  本次支出  | 申购时填写（注意前后空格） |
| 本次卖出数量 | 赎回时填写 |
| 本次收入 | 赎回时填写 |
|  累计份额  | 0（注意前后空格） |
|  累计支出  | 0（注意前后空格） |
|  本次交易后\n加权平均成本  | [PRICE]（含换行） |
| 本次赎回盈利 | 0 |
| 纳税年度是否错配 | 否 |
| 境外纳税年度 | [YEAR] |
| 对应内地纳税年度 | [YEAR] |

### Sheet 6: 税款计算表
自由格式表格，包含:
- 汇率日及USD/HKD汇率
- 股息分项（原始金额→人民币→应纳→预扣→实际应纳）
- 利息分项
- 财产转让分项（含"同年度盈亏互抵"标注）
- 税款合计

### Sheet 7: 透视表
交叉汇总表，包含股息、利息、股票交易、基金交易的分类汇总。
