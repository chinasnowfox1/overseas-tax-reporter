# 标准工作流程

```
用户: "我要报境外税"
        │
        ▼
  ┌─────────────────┐
  │ Phase 0: 确认年度 │
  │ 纳税年度(202X)    │
  └────────────────┘
           │
           ▼
  ┌─────────────────┐
  │ Phase 1: 收集素材 │
  │ - 券商列表        │
  │ - PDF/Excel路径   │
  │ - PDF密码         │
  └────────┬────────┘
           │
           ▼
  ┌─────────────────┐
  │ Phase 2: 税法科普 │
  │ - 收入类型        │
  │ - 税率20%         │
  │ - 盈亏互抵规则    │
  └────────┬────────┘
           │
           ▼
  ┌─────────────────┐
  │ Phase 3: 数据解析 │
  │ parser.py         │
  │ - 富途PDF解析     │
  │ - 其他券商适配    │
  └────────┬────────┘
           │
           ▼
  ┌─────────────────┐
  │ Phase 4: 税务计算 │
  │ calculator.py     │
  │ - 盈亏互抵        │
  │ - 预扣抵免        │
  └────────┬────────┘
           │
           ▼
  ┌─────────────────┐
  │ Phase 5: 输出Excel│
  │ exporter.py       │
  │ 7-sheet律师模板   │
  └────────┬────────┘
           │
           ▼
  ┌─────────────────┐
  │ Phase 6: 用户校验 │
  │ 检查关键数据      │
  └─────────────────┘
```

## 每个阶段的关键输入/输出

| 阶段 | 输入 | 输出 |
|------|------|------|
| 0 | 用户口述 | 纳税年度 |
| 1 | 用户文件路径 | 素材清单 |
| 2 | 无 | 税法摘要 |
| 3 | PDF目录 | 结构化交易数据 |
| 4 | 交易数据 + 汇率 | 税务计算结果 |
| 5 | 计算结果 | 7-sheet Excel |
| 6 | Excel | 确认或修正 |
